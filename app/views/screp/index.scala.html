About SCREP

SCREP is a StarCraft: Brood War replay parser that extracts detailed information from replay files.

Supported Features:
  • Player information
  • Game settings
  • Map details
  • Commands and actions
  • Game statistics
  • Both modern and legacy formats
Output Formats:
  • Full Parse: Complete JSON data
  • Overview: Human-readable summary
API Access

You can also access SCREP programmatically:

  • POST /screp/api/parse - Parse replay
  • POST /screp/api/overview - Get overview
  • GET /screp/health - Health check
